Loading and preprocessing the data

To load in the csv file I used the read.csv() function. I then used the transform() function to convert the date field from a factor to a date type.

activity <- read.csv(file="activity.csv")
activity <- transform(activity, date = as.Date(date))

What is mean total number of steps taken per day?

Once the data is in the correct format I can now create some exploratory graphs using the ggplot() function in the ggplot2 package. This needs to be loaded into R using the library() function.

library(ggplot2)
aggData <- aggregate(steps ~ date, data=activity, FUN=sum)
ggplot(aggData, aes(steps)) + geom_histogram(binwidth=600)

The mean and median can be calculated using the aggData variable we created and the mean() and median() functions as follows;

Imputing missing values

Here is the total number of NA values in the dataset.

sapply(activity, function(x)sum(is.na(x)))
##    steps     date interval 
##     2304        0        0

Begin by copying the activity data frame into a new varible called activity2. We can fill the missing values by using the 5 minute interval mean. We create a new column in the activity2 data frame that will hold the interval means and then replace the NA values in the steps column with these means. We can see using the sapply function that the NA are gone.

activity2 <- activity
activity2$meanInterval <- ave(activity$steps, activity$interval, FUN=function(x) mean(x, na.rm=T))
activity2$steps[is.na(activity2$steps)] <- activity2$meanInterval
## Warning: number of items to replace is not a multiple of replacement
## length
sapply(activity2, function(x)sum(is.na(x)))
##        steps         date     interval meanInterval 
##            0            0            0            0

New histogram using the filled in values.

aggData2 <- aggregate(steps ~ date, data=activity2, FUN=sum)
ggplot(aggData2, aes(steps)) + geom_histogram(binwidth=600)

The mean and median can be calculated using the aggData2 variable we created and the mean() and median() functions as follows;
